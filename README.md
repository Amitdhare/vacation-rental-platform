# Airbnb Clone

A full-stack Airbnb Clone web application built using Node.js, Express.js, MongoDB, and EJS templating engine with Server-Side Rendering (SSR) architecture.

## Features

* Add and Manage Listings
* Upload Property Images
* Responsive Design
* MongoDB Database
* Review Sysstem

## Tech Stack

* Frontend: EJS Template  HTML5
* Backend: Node.js + Express.js
* Database: MongoDB
* Styling: CSS / Bootstrap / Tailwind
